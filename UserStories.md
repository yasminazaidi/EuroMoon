## User Stories 

### Passagier registreren

Als gebruiker
wil ik een passagier kunnen registreren
zodat ik passagiers kan toevoegen aan het systeem.

### Criteria
•	Naam, achternaam, rijksregisternummer en geboortedatum worden gevraagd
•	Alle verplichte velden moeten ingevuld zijn
•	De passagier wordt correct opgeslagen

## Reis aanmaken

Als gebruiker wil ik een reis kunnen aanmaken
zodat ik treinreizen kan plannen.

### Criteria
Vertrek- en aankomstation worden ingevoerd
Datum en tijd worden opgeslagen
De reis wordt toegevoegd aan de lijst

##Ticket verkopen

Als gebruiker wil ik een ticket kunnen verkopen zodat een passagier kan meereizen.

### Criteria
Ticket wordt enkel verkocht als er plaats is
Ticket is gekoppeld aan een passagier
Ticket is gekoppeld aan een reis

### Structuur van het project

Als ontwikkelaar
wil ik een duidelijke projectstructuur
zodat de code overzichtelijk en onderhoudbaar is.

 criteria
•	Er is een app package voor de logica en menu’s
•	Er is een model package voor de klassen
•	Klassen zijn logisch opgesplitst
