## Euromoon – Java Trein Applicatie

Euromoon is een fictieve treinmaatschappij die internationale treinreizen binnen Europa organiseert.
Dit project is een Java command line applicatie **proof of concept**.



### Functionaliteiten

Bij het starten van de applicatie krijgt de gebruiker een menu met volgende opties:
1.	Registreren van een passagier
2.	Aanmaken van een reis
3.	Koppelen van een trein aan een reis
4.	Verkopen van een ticket
5.	Afdrukken van een boardinglijst (.txt)
6.	Stoppen van het programma



### Boardinglijst

Bij het afdrukken van de boardinglijst wordt automatisch een tekstbestand (.txt) aangemaakt.
•	Bestandsnaam:
Vertrek_Aankomst_Datum.txt
•	Inhoud:
overzicht van alle passagiers met een ticket voor de gekozen reis


### Structuur
•	app
Bevat Main.java met het keuzemenu en gebruikersinteractie
•	model
Bevat alle classes zoals Persoon, Passagier, Reis, Ticket, Trein en Locomotief

Alle belangrijke classes en methoden zijn voorzien van Javadoc.
